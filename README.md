## 大学生涯

### 为什么不选C

大一上的时候，学院的副院长给我们上C语言课，我第一次接触到了编程的乐趣，那个时候在命令行中打出 "Hello World" 觉得不可思议，然后改掉字符串，改成了 "XXX 是SB",第一时间分享给了我的 XXX 好兄弟。后来学到了指针的章节，那时觉得相当晦涩难懂，随后的课程就慢慢跟不上了，离C越来越远。或许是对C语言不够喜欢吧...

### 为什么选Java

大一下的时候，开设了一门面向对象程序设计课程，实际上就是Java，哇，当时觉得 Java 没有像 C语言满天飞感觉就像是来拯救我的职业生涯一般。学到后来，指针的概念是逃不掉的，计算机四大名著的知识，该学还得学。
